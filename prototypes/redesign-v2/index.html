05 · Editor · input pane

Input as a real editor surface, kept quiet.

Pane header has only the title and a context label. No floating chip toolbar. The footer offers a single discreet + snippet trigger that opens a command palette — same gravity as ⌘K. Code body has an explicit gutter, syntax tinting, active-line highlight, parse status.

10 · States

The happy path is the rare path.

Empty library, render error, chart-not-found, parse-error ribbon. All consume the same tokens and treat error tone with ruby (not red), warning tone with lemon.
